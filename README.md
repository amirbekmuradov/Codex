# Something Generator

A tiny Python CLI that prints one (or many) ideas for what to build next.

## Requirements

- Python 3.8+

## How to run

From the repository root:

```bash
python3 something.py
```

## Useful options

Print multiple ideas:

```bash
python3 something.py --count 3
```

Make output reproducible with a seed:

```bash
python3 something.py --seed 42
```

Show full CLI help:

```bash
python3 something.py --help
```
